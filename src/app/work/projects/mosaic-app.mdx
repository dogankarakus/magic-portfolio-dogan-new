## Overview

In this project, I implemented new features and enhanced existing ones for an existing Spotify-like audio streaming application, integrated with a physical music player. The app allows seamless music playback from multiple streaming APIs like Spotify, Tidal, Apple Music, and others, delivering an enhanced audio experience to users.

## Key Features

- **User Onboarding Flow**: Developed a structured onboarding process, guiding over 1K+ users through the app setup seamlessly.
- **Settings Screen for Embedded Display**: Enabled users to interactively configure device settings using an intuitive UI.
- **Reusable Architecture**: Ensured scalability and maintainability through clean architecture principles.

## Technologies Used

- **Flutter & Dart**: For cross-platform development.
- **BLoC Architecture**: For state management.
- **GraphQL**: For efficient data fetching and interactions.

## Challenges and Learnings

Creating onboarding pages with high reusability for future implementations required applying advanced OOP principles while adhering to BLoC architecture guidelines. 

## Outcome

The improved onboarding and settings management resulted in a smoother user experience and increased adoption rates.
