## Overview
In this project, I implemented new features and enhanced existing ones for an existing Spotify-like audio streaming application, integrated with a physical music player. The app allows seamless music playback from multiple streaming APIs like Spotify, Tidal, Apple Music, and others, delivering an enhanced audio experience to users.

## Key Features

- **Audio Session Initialization**: Ensured smooth playback with native UI controls and device-level integration.
- **Device Standby Mode**: Implemented a feature allowing users to disable IoT device connections when not in use.
- **Improvements and bug fixes**: Improved existing app features based on client feedback.

## Technologies Used

- **Flutter & Dart**: For app development.
- **Rust**: For handling device connection.

## Challenges and Learnings

Integrating audio session using native OS interface was challanging due to lack of Flutter packages out there, but I managed to complete the implementation by filing a bug report with my senior to the package developers.  

## Outcome

The implemented features improved user control and ensured a more reliable and flexible audio streaming experience.
