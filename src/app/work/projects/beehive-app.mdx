## Overview
In this project, I owned the design and implementation of the core gaming interfaces and the in-app purchase system. The application delivers daily word puzzles powered by dynamically generated, AI-tailored content.

## Key Features

- **Dynamic UI & Gameplay**: Designed and implemented high-performance, engaging core game screens with fluid animations and responsive layouts..
- **In App Purchases**: Developed a secure and seamless in-app purchase infrastructure to handle premium content unlocks and subscriptions..
- **Feature Evolution and optimization**: Iterated on existing gameplay mechanics and features, translating user and client feedback into polished product updates.

## Technologies Used

- **Flutter & Dart**: For app development.
- **Firebase**: For handling no-SQL backend infrastructure.
