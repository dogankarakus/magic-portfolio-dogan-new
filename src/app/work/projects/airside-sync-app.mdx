## Overview

Airside Sync is a digital platform for reporting SNOWTAM according to ICAO's Global Reporting Format (GRF) guidelines.

## Key Features

- **Push notifications**: Implemented a real-time notification infrastructure by integrating FCM across client application and developing server-side delivery services in Go using the Firebase Admin SDK.
- **Backend development**: Built backend architecture in Go to broadcast real-time, low-latency notifications across the Flutter application seamlessly.
- **UI Architecture & Theming**: Refactored unorganized application styles into a unified theme framework, utilizing modular, reusable widgets and establishing full support for light and dark modes.
- **Client onboarding**: Onboarded two new airports, setting up their configurations and map data via OpenStreetMap, Docker, and a
custom CLI.


## Technologies Used

- **Flutter & Dart**: For app development.
- **Go & gRPC **: For backend development.
- **BLoC**: For state management.
- **Firebase Cloud Messaging**: For push notifications.
