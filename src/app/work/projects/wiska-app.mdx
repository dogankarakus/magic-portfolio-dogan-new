## Overview

This project involved developing a remote control app for managing spotlight positions efficiently. The focus was on enabling users to automate movement loops and interact with a physical remote unit seamlessly.

## Key Features

- **Spotlight Positioning System**: Developed a feature that allows users to control, save, and automate spotlight movement using BLoC architecture with the existing Rust backend.
- **Custom UI for Remote Control Unit**: Created UI widgets for a custom embedded display (400x300), connecting controller's physical buttons to the Flutter UI.   

## Technologies Used

- **Flutter & Dart**: For building the mobile and embedded UI.
- **BLoC Architecture**: For managing application state effectively.
- **Rust Backend**: For handling device connection via localhost.

## Challenges and Learnings

One of the biggest challenges was to handle application state for movement loops, where it's difficult to get device's position and next movement information efficiently. 

## Outcome

The developed app allowed for seamless control of spotlights, significantly improving usability for end users while enhancing automation capabilities.
