## Overview

Blindside is a social training platform connecting coaches and athletes in individual and team sports. It allows coaches to create exercises with videos and instructions, while athletes can access and participate in these exercises to enhance their training.

## Key Features

- **Video Upload/Download**: Integrated AWS S3 & CloudFront for efficient video storage and streaming.
- **Feature development**: Developed 20+ app screens using Riverpod, MVVM architecture, and a GraphQL backend.
- **Enhanced Search & Profile Screens**: Improved user experience through optimized UI and performance.
- **Firebase Dynamic Linking**: Enabled seamless app sharing through deep linking functionality.


## Technologies Used

- **Flutter & Dart**: For app development.
- **AWS S3 & CloudFront**: For video storage and delivery.
- **Riverpod**: For state management.
- **Firebase Dynamic Links**: For improved user acquisition and sharing.

## Challenges and Learnings

Implementing video upload/download functionality using AWS S3 & CloudFront was a complex challenge requiring in-depth knowledge of AWS's documentations.

## Outcome

The app improvements resulted in better content sharing, enhanced search functionality, and an overall improved user experience.
